# What is this?

Warhammer 40,000: Conquest custom card creator. Designed as a standalone application. 

# Dependencies

Python 3.9.7, PIL 11.3.0, tkinter 0.1.0.
Git should be used for version control, but it is not strictly necessary.
If you are familiar with Python, you should also set up a virtual environment, though if you have no program experience you do not need to do this.

# How to install

Before installing this program, check that you have Python 3.9.7 and git installed.

The repo will be placed on your desktop if you copy these commands in exactly.

Open the command terminal.

~~~
cd Desktop
git clone https://github.com/C-C-Coalback/Conquest-Card-Creator Conquest-Card-Creator
cd Conquest-Card-Creator
pip install -r requirements.txt
~~~

# How to run

Double-click the main.py file.
